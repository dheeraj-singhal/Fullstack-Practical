About Course

Front-end web development is everything involved in programing the user interface of a web application. Typically it refers to the Hypertext Markup Language (HTML), Cascading Style Sheets (CSS) and JavaScript portionof web site production as opposed to the database or server-side programming. It encompasses everything from building a simple page of HTML text to creating cpmplex, responsive HTML5 websites designed to be accessed via various different browsers, devices and screen sizes.
